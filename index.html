Ejemplos

Código para implementar estilos CSS a nivel global

* {box-sizing: border-box;}

Código para llamar a una clase definida con la palabra "class " dentro de una etiqueta HTML

.divider {border-bottom: 1px solid #888989; margin: 2px 0;}

Código para definir la opacidad del color, en este caso 0.5; Va de 0 a 1. Podemos reemplazar opacity por rgba en el background-color, donde a es alpha que también va de 0 a 1

* background-color: rgba(255,255,255,0.5);

Ejemplo de código para darle un estilo al body de una página web

* body { width: 100%; height: 100vh; background-color: #1b1b32; color: #f5f6f7; margin: 0; font-family: Tahoma; font-size: 16px; }

Código para curvar las esquinas de los cuadrados

* border-radius: 30px 25px 60px 12px;
